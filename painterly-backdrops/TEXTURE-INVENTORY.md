## Blend Mode Guide

**Multiply**: Use for darkening and adding depth (25-50% opacity)
**Overlay**: Use for balanced texture and color (20-50% opacity)
**Soft Light**: Use for subtle, refined blending (15-45% opacity)
**Screen**: Use for lightening and warm glows (10-35% opacity)
**Color Dodge**: Use for dramatic highlights (10-25% opacity)
**Darken**: Use for selective darkening (25-60% opacity)
**Color Burn**: Use for deep, rich effects (15-40% opacity)
